# 项目概述
项目旨在结合ABACUS与DeePKS计算框架，以孤立氢原子体系为研究对象，通过机器学习方法训练交换关联泛函，提升ABACUS计算能级的精度，使其更接近氢原子光谱的理论值。研究进展主要包括：系统学习了密度泛函理论及相关课程知识；掌握了ABACUS与DeePKS软件的使用、编译与配置；深入阅读了ABACUS源代码，理解了其自洽场迭代与DeePKS模块的实现逻辑；完成了对孤立氢原子体系不同泛函（PBE/HSE）的初步计算，并成功复现了DeePKS在水分子体系上的训练，验证了该方法提升计算精度的潜力。
计算结果表明，传统泛函对氢原子能级的描述仍有显著误差，而DeePKS方法在水分子体系上已展现出精度提升的有效性。目前存在的主要问题包括传统泛函的局限性、激发态计算的理论基础有待厘清，以及研究方法上需要进一步聚焦。下一步计划将改进科研方法，并考虑阅读DeePKS源码，尝试将氢原子解析解作为基准纳入损失函数重新设计，以更有效地训练模型。
# 项目进展
本项目采用ABACUS与DeePKS相结合的计算框架，以孤立氢原子体系为研究对象，训练交换关联泛函提高ABACUS能级计算精度，使其计算结果尽可能与氢原子光谱吻合。为实现该目标，本项目采用大晶胞模型近似模拟孤立氢原子体系，并基于ABACUS进行第一性原理计算，结合DeePKS学习真实氢原子光谱这一基准体系进行泛函训练，以期提升密度泛函理论在描述单原子及类似体系电子结构时的准确性与物理一致性。
经过半年多的研究学习，项目实施进展如下：
其一，完善理论知识储备。学习密度泛函理论，通过自学与学习《量子力学》《热力学与统计力学》等课程，掌握固体力学、量子力学的基本知识，从理论上把握如何通过自洽场迭代法求解周期性边界条件下晶胞体系的电子密度，从而把握ABACUS求解的原理，分析孤立氢原子体系能级求解的理论误差来源。
其二，熟悉使用ABACUS与deepks-kit的流程。输入文件的参数调整与输出文件的阅读和使用，如自洽场迭代（SCF）计算、RELAX计算，以及DeePKS数据集配置，输入文件参数配置等。
其三，完善编程知识。例如，补充cmake基础知识，了解项目的编译和构建过程。补充C++类实现，类继承，虚函数，预处理命令等知识，为阅读ABACUS源代码提供基础。
其四，阅读ABACUS源代码。各个模块实现功能，ABACUS的编译、运行流程，文件之间的关系。例如，阅读自洽场迭代（SCF）计算在ABACUS源代码的完整实现流程，了解ABACUS软件各个模块的实际调用关系、实现功能与运行流程，ESolver模块（以 ESolver_KS_PW 为例）的具体实现，以及DeePKS模块的组成，各个文件实现的内容。从代码层面把握ABACUS软件，从而把握孤立氢原子体系能级计算在ABACUS中的代码运行逻辑。
其五，编译ABACUS与deepks-kit等软件包并配置环境。安装并编译libxc等软件包，从而使ABACUS能进行杂化泛函计算，并支持机器学习算法。配置DeePKS环境，解决版本依赖问题，从而使Deepks-kit能与ABACUS交互训练交换关联泛函。
其六，用ABACUS计算孤立氢原子体系并分析计算结果，利用ABACUS+DeePKS计算。用不同精度的交换关联泛函（PBE\HSE）计算原子轨道基组与平面波基组（lcao\pw）体系，所得轨道能量与氢原子光谱有较大差别。针对孤立氢原子体系，有待进一步提升。利用ABACUS+DeePKS计算复现水分子算例，相较于预训练模型，训练结束后模型的误差明显降低。
阶段性成果：掌握了ABACUS求解理论基础，理解了ABACUS计算与DeePKS训练的实现机制；初步计算表明，传统泛函（PBE/HSE）对孤立氢原子能级的描述仍存在显著误差，而在水分子体系上，ABACUS+DeePKS训练后的模型精度较预训练模型有明显提升，验证了该方法在提升电子结构计算精度方面的可行性与潜力。
# 目前存在的问题
传统泛函（PBE/HSE）对孤立氢原子能级的描述仍存在显著误差，如果按照DeePKS以往的数据集和训练结果（以多帧较高泛函精度的数据集训练的模型，能够在提高交换关联泛函精度的同时缩减ABACUS计算时间）可能训练效果不明显，所以可以考虑阅读DeePKS源代码，重新设计损失函数项。
理论上如果交换关联泛函足够精确时，KS方程与氢原子定态薛定谔方程等价。然而，传统KS方程中的轨道能量缺乏严格的物理对应，且由于采用电子密度表达势函数，会引入自相互作用误差。ABACUS+DeePKS如何计算出激发态能量，尚欠缺一定的理论基础。当然这并不意味着经过机器学习训练后，ABACUS+DeePKS无法计算出氢原子激发态能量。
针对这些问题，由于氢原子体系具有解析精确的能级解，可作为理想的基准体系，将其作为常量写入损失函数，用来训练ABACUS生成的杂化泛函精度下孤立氢原子体系的结果，有希望训练出具有明显效果的模型。
